# Ficha rápida para D&D

## Sobre
Webapp para agilizar a criação de personagem de D&D 5ª edição

## Ingredientes
- Nextjs
- ChakraUI
- ContextAPI
- Paciência
- Unhas roídas de nervoso

## Features
- Poderá ser pública ou privada;
- Caso seja privada:
  - Login com Google;
- Auto-saving;
- Vincular ficha com DungeonMaster
- Internacionalização (pt-BR e en-US)
